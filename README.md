# Personal Portfolio Website

A modern, fully responsive personal portfolio website built with HTML5, CSS3, and JavaScript.

## Live Demo

🔗 **View Website:** [https://aswini-portfolio.netlify.app](https://aswini-portfolio.netlify.app)

## Features

- Modern dark theme with gradient accents
- Custom cursor animation
- Rotating text effect (Developer/Designer)
- Project showcase with filter buttons
- Testimonial slider
- Working contact form with validation
- 3D particle background
- Fully responsive for all devices
- Smooth scroll navigation

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Netlify (Hosting)
